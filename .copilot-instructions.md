# GitHub Copilot Instructions - dnd-5e-core
## Package Overview
Complete D&D 5e Rules Engine - UI-agnostic game logic
## Key Principles
1. **UI-Agnostic**: Return data, don't display it
2. **Use Generators**: `simple_character_generator()` not `Character(...)`
3. **Check Before Action**: `if char.can_cast(spell)` before casting
4. **Handle Errors**: Class requirements may cause ValueError
## Common Patterns
### Character Creation
```python
from dnd_5e_core.data.loaders import simple_character_generator
char = simple_character_generator(5, "human", "fighter", "Hero")
```
### Combat
```python
from dnd_5e_core.combat import CombatSystem
combat = CombatSystem(verbose=True)
combat.character_turn(char, alive_chars, alive_monsters, party)
```
### Equipment
```python
free_slot = next((i for i, item in enumerate(char.inventory) if item is None), None)
if free_slot is not None:
    char.inventory[free_slot] = item
    char.equip(item)
```
## Class Requirements
- Paladin: STR 13, CHA 13
- Monk: DEX 13, WIS 13
- Wizard: INT 13
