## 📦 Question 1: Is the egg-info folder necessary?

### Answer: **NO** ❌

The `dnd_5e_core.egg-info/` folder is **NOT necessary** for publication.

#### Explanations:

**This folder is:**
- ✅ Automatically generated by `pip install -e .` (installation in development mode)
- ✅ Contains temporary metadata for your local environment
- ✅ Already in `.gitignore` (line 21: `*.egg-info/`)
- ❌ Should **NEVER** be committed to Git
- ❌ Is **NOT** used during publication on PyPI

**During publication:**
- PyPI uses metadata from `pyproject.toml`
- The build creates `dist/` containing `.tar.gz` and `.whl` files
- These files in `dist/` are the ones uploaded to PyPI

**Files to ignore (already in .gitignore):**
```
*.egg-info/     ❌ Local metadata
dist/           ❌ Compiled distributions
build/          ❌ Temporary build files
__pycache__/    ❌ Python cache
*.pyc           ❌ Python bytecode
```
