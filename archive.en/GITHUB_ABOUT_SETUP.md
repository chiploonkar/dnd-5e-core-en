# Configuring the "About" Section on GitHub

## 📝 How to Configure

1. Go to your GitHub repository: `https://github.com/codingame-team/dnd-5e-core`
2. Click on the ⚙️ (Settings) icon to the right of the repository title
3. In the "About" section, configure the following items:

## 📄 Description

```
Complete D&D 5th Edition Rules Engine - Core game logic with 332 monsters, 319 spells, and full offline data. UI-agnostic Python package for building D&D applications.
```

## ⚖️ Check These Options

- ☑️ **Include in the home page**: Yes
- ☑️ **Releases**: Yes (displays versions)
- ☑️ **Packages**: No (unless you use GitHub Packages)
- ☑️ **Deployments**: No (unless you deploy)

## 📝 Social Preview

GitHub can generate a social preview image. To customize it:

1. Go to **Settings** > **Options** > **Social preview**
2. Click on **Edit**
3. Upload a 1280×640px image (optional)

Image suggestions:
- D&D logo with the package name
- Screenshot of your code/documentation
- Illustration of a 20-sided die with code
