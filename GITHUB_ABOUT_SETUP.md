# Configuration de la Section "About" sur GitHub

## 📝 Comment configurer

1. Allez sur votre dépôt GitHub: `https://github.com/codingame-team/dnd-5e-core`
2. Cliquez sur l'icône ⚙️ (Settings) à droite du titre du dépôt
3. Dans la section "About", configurez les éléments suivants:

## 📄 Description

```
Complete D&D 5th Edition Rules Engine - Core game logic with 332 monsters, 319 spells, and full offline data. UI-agnostic Python package for building D&D applications.
```

## 📝 Social Preview

GitHub peut générer une image de prévisualisation sociale. Pour la personnaliser:

1. Allez dans **Settings** > **Options** > **Social preview**
2. Cliquez sur **Edit**
3. Uploadez une image 1280×640px (optionnel)

Suggestions d'image:
- Logo D&D avec le nom du package
- Screenshot de votre code/documentation
- Illustration d'un dé à 20 faces avec du code
